# Hi there, I'm Muhammad Ans 👋

### Embedded Systems • IoT • Robotics • Firmware • Hardware Bring-up

I build practical embedded and IoT systems that connect **hardware, firmware, sensors, wireless communication, robotics, and software dashboards** into working products.
